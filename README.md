# AI PDF知识检索助手

## 1. 项目介绍

AI PDF知识检索助手是一个基于 Python 和 Streamlit 开发的轻量级网页应用。

用户可以上传一个 PDF 文件，输入关键词，系统会自动读取 PDF 全部内容，并从文档中检索出和关键词最相关的内容，帮助用户更快定位知识点。

这个项目适合初学者学习以下内容：

- Python 基础语法
- PDF 文本读取
- 简单文本清洗
- 关键词匹配检索
- Streamlit 网页应用开发

## 2. 功能说明

- 支持上传一个 PDF 文件
- 自动读取 PDF 的全部页面内容
- 对 PDF 文本进行基础清洗，减少多余空格和换行影响
- 支持输入关键词进行检索
- 从 PDF 中提取和关键词最相关的句子
- 展示 AI 回答内容
- 展示参考原文片段
- 显示参考内容所在页码
- 对关键词进行高亮显示，方便快速查看

## 3. 技术栈

- Python
- Streamlit
- PyPDF2
- re（Python 内置正则表达式库）
- io.BytesIO（Python 内置字节流处理）

## 4. 安装步骤

### 4.1 克隆或下载项目

如果你已经有项目文件，可以直接进入项目目录。

```bash
cd '/Users/lby/aj，python'
```

### 4.2 安装依赖

建议先安装项目需要的依赖：

```bash
pip install streamlit PyPDF2
```

如果你使用的是 Python 3 环境，也可以这样安装：

```bash
pip3 install streamlit PyPDF2
```

## 5. 运行方法

进入项目目录后，运行下面命令：

```bash
streamlit run main.py
```

运行成功后，终端通常会显示本地访问地址，例如：

```bash
http://localhost:8501
```

然后你可以在浏览器中打开这个地址，进入系统页面。

使用流程如下：

1. 上传一个 PDF 文件
2. 在输入框中输入关键词
3. 点击“开始检索”按钮，或者按回车提交
4. 查看 AI 回答、参考原文片段和页码定位

## 7. 项目结构

当前项目结构如下：

```bash
.
├── main.py
├── requirements.txt
├── generate_plot.py
├── result.png
└── README.md
```

各文件说明：

- `main.py`
  项目的主程序文件，负责 PDF 上传、文本读取、关键词检索、结果展示等核心功能。

- `requirements.txt`
  项目依赖文件，目前你可以把需要的依赖写入这里，方便后续统一安装。

- `generate_plot.py`
  一个独立的 Python 示例脚本，用于生成随机数折线图，与 PDF 检索主功能无直接关系。

- `result.png`
  由 `generate_plot.py` 生成的图片文件。

- `README.md`
  项目说明文档，用于介绍项目功能、安装方法和使用方式。
